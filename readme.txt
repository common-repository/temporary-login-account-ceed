== Description ==

Temporary Login offers you to create WordPress login accounts with different roles and you can set expired times for those accounts. You can extend the time limit with the automatic mail feature.

== Frequently Asked Questions ==

= 1. Do I need send username & password manually? =

No need to send manually.It will be mailed once you created the credentials in settings.

= 2. Can I delete Temporary user manually? =

Yes, you can delete manually.
